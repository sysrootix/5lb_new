# Сертификаты для Balance API

## Инструкция

1. Поместите файл сертификата `terehin_n.cloud.mda-medusa.ru.p12` в эту директорию
2. Убедитесь, что файл имеет правильные права доступа:
   ```bash
   chmod 600 terehin_n.cloud.mda-medusa.ru.p12
   ```

## Важно!

⚠️ **НЕ коммитьте** файлы `.p12` в репозиторий!
Сертификаты должны храниться отдельно и добавляться на сервере вручную.

## Пароль сертификата

Пароль для сертификата: `000000000` (9 нулей)
Настройте его в переменной окружения `CERT_PASSWORD`
