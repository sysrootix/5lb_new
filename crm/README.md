# 5LB CRM System

## Backend
Located in `crm_backend`.
Port: 5000

### Setup
1. `cd crm_backend`
2. `npm install`
3. `npx prisma generate`
4. `npm run seed` (to create root admin)
5. `npm run dev`

## Frontend
Located in `crm_frontend`.
Port: 5001

### Setup
1. `cd crm_frontend`
2. `npm install`
3. `npm run dev`
